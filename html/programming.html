Programación desarrollada

Se implementan diferrentes lenguajes, python, micro python, html, css, y javascript, principalmente. Además, también se pretende poder almacenar grabaciones y fotografías, según así se requiera, con el robot, de entre las aplicaciones está manejar archivos de tipo cvs.

Python:

Usado principalmente para diseñar la base de datos que permitira manejar los archivos cvs, en este tipo de formato se almacenara la información adquirida de la Araña Robot, entre la que está la recolectada por el sensor de humedad y temperatura, y por el sensor de ultrasonido, la distancia cuando se encuentre cerca o este acercando a paredes u obstaculos. Al ser Python un lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning (ML). Desarrollar códigos utilizando Python es eficiente, siendo fácil de aprender, también permite la ejecución en muchas plataformas diferentes. Así, el software Python, se descarga gratis, se integra bien a todos los tipos de sistemas, y aumenta la velocidad del desarrollo.

MicroPython:

Usado para programar la araña y todo el funcionamiento de la misma, desde los servomotores que ejecutan los movimientos de las extremidades, el microprocesador ESP32 Cam, la ESP32 clásica, que componen entre ambas los cerebros del robot, así mismo, a través de estos los sensores y demás funcionamientos. A parte de ser una versión adaptada de Python para microcontroladores, es un lenguaje interpretado, o, no se compila para ejecutarse sino que se interpreta durante su ejecución, siendo una heerramienta útil para proyectos que requieran controlar las entradas/salidas (inputs/outputs) de un microcontrolador, tomando lecturas de señales analógicas y digitales, generando señales PWM, control de motores y servomotores, entre otros. Además, en microcontroladores como la ESP32 también es posible conexiones de Red vía WiFi y Bluetooth.

HTML:

Usado para desarrollar una pagina web y el contenido en esta. Es básicamente donde se ha escrito y estructurado lo que se quería mostrar en este sitio, todo lo que está aquí sea informativo, imagenes, videos, entre otros recursos que se encuentren, se realizo en html. Por medio de este Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language, se empieza con la parte más básica que compone el diseño de una página web. Pues, define el significado y la estructura del contenido web. Aunque "Hipertexto" se refiere a los vinculos que conectan páginas web entre sí, ya sea dentro de un único sitio o varios, también se debe tener en consideración que al subir contenido a Internet y vincularlo a las páginas creadas por otras personas, empieza a ser un participante activo en la World Wide Web (Red Informática Mundial).

CSS:

Diseña la pagina Web y el aspecto que esta tendrá, la forma en que se verá, ajustando colores, tipos de letra, margenes, algunas transiciones, y en general la apariencia que se le pretenda dar. Una aclaración, CSS significa (Hojas de estilo en cascada) y parten de la idea concreta de aplicar estilos a uno o varios documentos (HTML, páginas webs) de forma más automática y masiva. Se le denomina de esta forma porque se lee, procesa y aplica el código desde arriba hacia abajo, manteniendo una secuencia denominada herencia, y en el caso de existir ambigüedad,entendido como código que se contradice, existen normas para resolver dichas ambigüedades.

JavaScript:

Permite a los sitios web responder con acciones del usuario, clics, desplazamientos, y comandos de teclado, para proporcionar una experiencia interactiva. Ejemplos destacados e incluido en este sitio, incluye menús desplegables, otros son carruseles de imágenes, y validación de formularios. Por lo tanto, en este lenguaje de programación o de secuencias de comandos, se implementará funciones "complejas" en la página web, para que de esta forma la página web haga algo más que mostrar información, sacando ventaja de sus aplicaciones,ya sea mostrar oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc., es necesario usar JavaScript en el dinamismo de las páginas.
